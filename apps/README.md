# Apps

Each product lives in its own folder so the repo stays easy to scan and each app can be extracted later.

- `jagsrajkitchen`: Raja Kitchen restaurant site.
- `ailearnraja`: AI learning/interview prep app.
- `forma-fitness`: Forma Fitness landing/product app.
- `genzstylelearn`: Gen Z style learning app.
- `resetmindproj`: Reset Mind project app.
- `apple-notes-clone`: Apple Notes-style notes app.
- `ai-ascend-academy`: AI Ascend Academy learning app.
- `capture-thoughts`: Thought capture and reflection app.
- `guide-learn-ai`: Guided AI learning app.
- `speakpracticetamil`: Tamil speaking practice app.
- `desievents`: Desi events discovery app.
- `techblograja`: Tech blog project.
- `mathraja`: MathDreams elementary math learning app.
- `schoolosagent`: Campus AI school operating system dashboard.
- `serverdashboard`: Stage server/dev dashboard.
- `jobsopswithraja`: JobOps job application command center.
- `intervqans`: StudyDeck interview question and answer prep app.

Add future projects as `apps/<project-name>` and register them in the root `package.json` scripts plus the portal cards in `src/routes/index.tsx`.
